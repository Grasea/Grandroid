# Grandroid
Android open source library


#How to import Grandroid to project
```
dependencies {
    compile 'com.grasea:grandroid:1.0.3'
}
```
#Only import database
```
repositories {
    maven {
        url 'https://dl.bintray.com/ch8154/maven'
    }
}

dependencies {
    compile 'com.grasea:grandroid-database:1.0.0'
}
```
